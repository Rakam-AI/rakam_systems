# Getting Started

## Set up the development environment

```bash
git clone git@github.com:Rakam-AI/rakam_systems.git
cd rakam_systems
python3 -m venv venv
source venv/bin/activate
pip install -e core/ -e ai-components/agents/ -e ai-components/vector-store/ -e tools/ -e cli/
```
