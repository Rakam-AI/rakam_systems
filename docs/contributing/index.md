# Contributing

Rakam Systems is open source and welcomes contributions. Whether you fix a bug, improve documentation, or propose a new feature, your help is appreciated.

- **Repository:** [github.com/Rakam-AI/rakam_systems](https://github.com/Rakam-AI/rakam_systems)
- **Issues:** [github.com/Rakam-AI/rakam_systems/issues](https://github.com/Rakam-AI/rakam_systems/issues)

## How to contribute

1. Fork the repository and create a branch from `main`.
2. Make your changes following the [Development](./development.md) and [Documentation](./documentation.md) guidelines.
3. Open a pull request against `main` on `Rakam-AI/rakam_systems`.

All contributors are expected to follow the [Code of Conduct](./code-of-conduct.md).
